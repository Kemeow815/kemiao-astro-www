## Editor & Terminal

- [VS Code](https://code.visualstudio.com/) 是我使用频率最高的编辑器，从它的第一个版本开始用起，堪称“铁粉”💻🎉。我写过的几个插件 [open html in browser](https://marketplace.visualstudio.com/items?itemName=coderfee.open-html-in-browser)、[vscode wxml](https://marketplace.visualstudio.com/items?itemName=coderfee.vscode-wxml)
- [Neovim](https://neovim.io/) 主要是作为终端编辑器使用，不到万不得已不会主动去用，确实学习成本比较高⌨️🤓
- [JetBrains](https://www.jetbrains.com/) 主要在公司用，因为公司的项目比较大，JetBrains 的重构功能深得我心🏢💡
- [iTerm2](https://iterm2.com/) 使用时间最长的终端，很喜欢，自定义程度比较高⚙️🖥️
- [Kitty](https://sw.kovidgoyal.net/kitty/) 这是最近开始使用的终端，感觉比 iTerm2 更轻量，配置简单🐱🌟
- [Oh My Zsh](https://ohmyz.sh/) 配合以上终端使用，配置简单，功能丰富，很赞💻🌈
- [Obsidian](https://obsidian.md/) Markdown 笔记工具，用坚果云做全平台同步功能，满足日常需求📝☁️

## Browser

- [Chrome](https://www.google.com/chrome/) 使用时间最长，从 52.x 版本开始用起，帮我摆脱 IE，我也是它的“铁粉”🌐🚀
- [Arc](https://arc.net/) 2023 年炙手可热的浏览器，用习惯之后很丝滑。不过 Windows 版本泛善可陈🔥💻
- [Firefox](https://www.mozilla.org/en-US/firefox/) 偶尔用用，越来越像 Chrome🦊🤔

## Desktop App

- [Vivo 办公套件](https://pc.vivo.com) Vivo 出品的跨设备协同套件，配合我的 Vivo X80 食用很爽，可以摸鱼📱🐟
- [坚果云](https://www.jianguoyun.com/) 全平台网盘，主要存一些证件照啥的，比较方便，有免费版📁☁️

### Windows

- [Ditto](https://ditto-cp.sourceforge.io/) 剪切板管理工具，比 Windows 自带的功能更加丰富，推荐 [Dracula](https://draculatheme.com/ditto) 主题✂️🧛‍♂️
- [PowerToys](https://github.com/microsoft/PowerToys) 微软官方出品的系统管理工具，提供窗口管理、文件管理、键盘管理、系统管理等功能⚙️💼
- [Pixpin](https://pixpinapp.com/) 截图工具，类似 Snipaste，不过内置 OCR，我是从 Snipaste 迁移过来的📸🖼️

### macOS

- [Raycast](https://www.raycast.com/) 效率类工具，提供 AI 功能、剪切板、快捷键等功能，不过 Pro 版本略贵。我最喜欢的软件之一⚡💻
- [HiddenBar](https://github.com) 菜单栏整理工具🔍📊
- [Shottr](https://shottr.ren/) 截图工具，提供的 OCR 我用过很多次，非常方便📸🔠
- [Rectangle](https://rectangleapp.com/) 窗口管理工具，记住几个快捷键，效率提升杠杠滴🔲⌨️
- [Keka](https://www.keka.io/) 解压缩工具💼🗄️
- [AltTab](https://alt-tab-macos.netlify.app/) 窗口切换工具🔄🖥️
